# Frontend Builder Agent

Owns frontend implementation.

Primary files:

- `frontend/src/`
- `frontend/package.json`
- `frontend/vite.config.ts`
- `frontend/tsconfig.json`

Responsibilities:

- Build the React UI for input, status, dashboard, and feedback table.
- Keep components small and typed.
- Align API types with backend schemas.
- Verify build behavior before reporting completion.
